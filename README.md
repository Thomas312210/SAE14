# Site Web : Pedri González

Bonjour. Voici le dépôt de mon projet de développement web (R1.15). J'ai choisi de créer un site biographique sur Pedri, le milieu de terrain du FC Barcelone, car c'est un joueur que j'apprécie beaucoup.

## Voir le site en ligne

Le site est accessible directement sur GitHub Pages en cliquant ici :
(https://github.com/Thomas312210/SAE14)

## De quoi parle le site ?

J'ai structuré le site en 5 pages pour couvrir différents aspects, tout en gardant une charte graphique aux couleurs de son club et de l'Espagne :

1.  **Accueil (`index.html`)** : La page principale qui présente le sujet et sont actualité en bas de page.
2.  **Qui est Pedri ? (`Qui-est-Pedri.html`)** : Une page plus personnelle sur son enfance et ses origines aux Canaries.
3.  **Sa Carrière (`Carriere.html`)** : Son parcours pro, de ses débuts à Las Palmas jusqu'au Barça.
4.  **Palmares (`Palmares.html`)** : La liste de ses trophées (Golden Boy, titres en Liga, etc.).
5.  **À Propos (`a-propos.html`)** : Une page pour me présenter (l'auteur du site) et expliquer le contexte du projet.

## Ce que j'ai utilisé (Technique)

Pour réaliser ce site et respecter les consignes, j'ai utilisé :

* **HTML5 & CSS3** : Pour la structure et le style.
* **Mise en page** : J'ai utilisé **Flexbox** pour placer mes éléments.
* **Responsive** : Le site s'adapte bien sur ordinateur, tablette et mobile.
* **Animations** : J'ai ajouté des animations CSS (par exemple au survol) pour rendre le site plus vivant.
* **Accessibilité & Standards** : J'ai fait attention à ce que le code soit valide (W3C) et accessible (normes WCAG).
